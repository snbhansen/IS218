# IS218
Student repository - Group 15

Dette webkartet viser brannalarmsentraler og trafikkulykker i Norge fra de siste fem kalenderårene som har ført til personskader eller større materielle skader. Datasettene er hentet fra GeoNorge. Brukere kan klikke på punktene for å se informasjon om hvert objekt.

**Demo**
- Klikk på blå punkter (brannalarmsentraler) for å vise navn og sted.
- Klikk på røde punkter (trafikkulykker) for å vise årsak (ulykkeskode/klassifisering), dato og skadegrad.

**Teknisk oppsett**
- Kartbibliotek: MapLibre GL JS v5.16.0
- Bakgrunnskart: Positron GL Style
- Dataformat: GeoJSON

**Datakatalog**
- Datasett: Brannalarmsentraler og trafikkulykker fra de siste fem kalenderårene
- Kilde: GeoNorge (https://kartkatalog.geonorge.no/)
- Bearbeiding: Lastet ned fra GeoNorge, bearbeidet i QGIS og eksportert som punktlag
