# IS218
Student repository - Group 15

Dette webkartet viser brannalarmsentraler og alle trafikkulykker de siste 5 kalenderårene med personskader eller større materielle skader i Norge. Brukere kan klikke på punktene for å se informasjon om hvert objekt.

**Demo**
- Klikk på blå punkter for brannalarmsentraler → viser navn og sted.
- Klikk på røde punkter for trafikkulykker → viser årsak, dato og skadegrad.

**Teknisk stack**
Kartbibliotek: MapLibre GL JS v5.16
Basemap: Carto Positron GL
Dataformat: GeoJSON

**Datakatalog**
Datasett: Brannalarmsentraler & Trafikkulykker de siste 5 kalenderårene
Kilde: GeoNorge (https://kartkatalog.geonorge.no/)
Bearbeiding: Lastet ned fra QGIS, brukt som punktlag
