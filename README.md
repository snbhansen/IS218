# IS218
Student repository - Group 15

Dette webkartet viser trafikkulykker de siste 5 kalenderårene med personskader eller større materielle skader i Norge. Datasettet er hentet fra GeoNorge. Brukere kan klikke på punktene for å se informasjon om hvert objekt.

**Demo**
- Klikk på røde punkter for trafikkulykker → viser årsak (ulykkeskode/klassifisering), dato og skadegrad.

**Teknisk stack**
Kartbibliotek: MapLibre GL JS v5.16.0
Basemap: Positron GL Style
Dataformat: GeoJSON

**Datakatalog**
Datasett: Trafikkulykker de siste 5 kalenderårene
Kilde: GeoNorge (https://kartkatalog.geonorge.no/)
Bearbeiding: Lastet ned fra QGIS, brukt som punktlag
